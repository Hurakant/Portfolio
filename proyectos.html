Firmware emulador de mando(Raspberry pi pico 2 W)

Firmware desarrollado con pico-sdk 2.2.0 para la Raspberry Pi Pico 2 W que implementa un dispositivo HID (Human Interface Device) sobre USB. El microcontrolador se enumera ante el sistema operativo como un gamepad estándar, sin necesidad de drivers adicionales, logrando compatibilidad nativa con Windows y Android. La comunicación se gestiona mediante descriptores USB personalizados que definen el mapa de botones y ejes analógicos enviados por el bus serial USB.

Aplicación de administración de inventario

Aplicación de escritorio para administración de inventario. Permite visualizar artículos en una grilla, buscar por cualquier parámetro de la tabla y aplicar filtros de visibilidad. Cada artículo cuenta con una vista de detalle con soporte para múltiples imágenes.

Incluye gestión completa de artículos, categorías, marcas e imágenes. Toda la información se persiste en una base de datos relacional en SQL Server, con consultas y operaciones CRUD manejadas desde la capa de datos de la aplicación.
